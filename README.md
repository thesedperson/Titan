# Titan

**Titan** is a fast and efficient subdomain enumeration and reconnaissance tool designed for security researchers and bug bounty hunters.

## Features

* **Fast Enumeration:** Multi-threaded scanning for rapid results.
* **Output Management:** Auto-saves results to easy-to-parse text or JSON files.
* **Lightweight:** Minimal dependencies for easy deployment.

```bash
git clone [https://github.com/thesedperson/Titan.git](https://github.com/thesedperson/Titan.git)
